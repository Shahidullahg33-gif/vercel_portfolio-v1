# Public Assets

This folder contains all the static assets for your portfolio. Replace these placeholder files with your actual content.

## Required Images

### Profile & Branding
- `avatar.jpg` - Your profile picture (400x400px recommended)
- `og-image.jpg` - Open Graph image for social sharing (1200x630px)
- `favicon.ico` - Website favicon

### Projects
Add your project screenshots to the `projects/` folder:
- `ecommerce.jpg` - E-commerce project screenshot
- `taskmanager.jpg` - Task manager project screenshot  
- `weather.jpg` - Weather app screenshot
- `blog.jpg` - Blog platform screenshot
- `fitness.jpg` - Fitness app screenshot

### Documents
- `resume.pdf` - Your latest resume/CV

## Image Guidelines

### Profile Picture (`avatar.jpg`)
- **Size**: 400x400px minimum
- **Format**: JPG or PNG
- **Style**: Professional headshot, good lighting
- **Background**: Clean, preferably neutral

### Project Screenshots
- **Size**: 1200x800px recommended
- **Format**: JPG or PNG
- **Quality**: High resolution, clear interface
- **Content**: Show the main interface/dashboard

### Open Graph Image (`og-image.jpg`)
- **Size**: 1200x630px (Facebook/LinkedIn standard)
- **Format**: JPG or PNG
- **Content**: Your name, title, and branding
- **Text**: Large, readable fonts

## Optimization Tips

1. **Compress images** using tools like TinyPNG or ImageOptim
2. **Use WebP format** for better compression (Next.js will auto-convert)
3. **Consistent sizing** for project images creates better visual harmony
4. **Alt text** is automatically handled by the components

## Updating Images

1. Replace the placeholder files with your actual images
2. Keep the same filenames or update the references in the data files
3. Restart the development server to see changes
